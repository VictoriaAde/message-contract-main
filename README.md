# Message Contract

## Description

A contract that that sets name and gets name

## Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/)
- [MetaMask](https://metamask.io/) browser extension installed

### Installation

1. Clone the repository.
2. cd message-frontend
3. yarn install
4. yarn dev
